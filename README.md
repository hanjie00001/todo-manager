# 个人待办事项管理器

[![Python Version](https://img.shields.io/badge/python-3.6%2B-blue)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/license-MIT-green)](LICENSE)

一个简洁实用的 **命令行待办事项管理工具**，使用 Python 编写，支持 JSON 数据持久化存储。

本项目适用于 Python 初学者练习 **数组/链表操作、文件读写、排序算法及简单菜单设计** 等核心编程概念。

## 功能特性

### 核心功能

| 功能 | 说明 |
|------|------|
| ✅ **添加待办事项** | 输入内容、选择优先级（高/中/低）、设置截止日期 |
| 📋 **查看待办列表** | 支持按优先级或截止日期排序显示 |
| ✔️ **标记已完成** | 按 ID 标记事项为已完成状态 |
| 🗑️ **删除待办事项** | 可删除所有已完成事项，或按 ID 单独删除（含二次确认） |

### 扩展功能

| 功能 | 说明 |
|------|------|
| 🔍 **状态筛选** | 全部 / 未完成 / 已完成 三种视图切换 |
| 📊 **统计概览** | 显示总数、完成数、未完成数及完成率 |
| 💾 **数据持久化** | 自动保存为 `todos.json` 文件，程序重启不丢失 |
| ⌨️ **取消操作** | 添加过程中可按 `Ctrl+C` 安全取消 |

## 安装与运行

### 方法一：直接运行

```bash
# 克隆仓库
git clone https://github.com/hanjie00001/todo-manager.git
cd todo-manager

# 运行程序
python todo_manager.py
```

### 方法二：添加至系统 PATH

```bash
# 赋予可执行权限（Linux / macOS）
chmod +x todo_manager.py

# 添加到 shell 配置文件（~/.bashrc 或 ~/.zshrc）
alias todo="python /绝对/路径/todo_manager.py"

# 重新加载配置
source ~/.bashrc

# 任意位置运行
todo
```

程序启动后会自动在同目录下创建 `todos.json` 数据文件，无需手动初始化。

### 1. 主菜单

程序启动后显示主菜单，输入数字选择功能：

```
          ★ 个 人 待 办 事 项 管 理 器 ★

  ┌─ 主菜单 ───────────────────────────┐
  │  1. 添加待办事项                     │
  │  2. 查看待办事项                     │
  │  3. 标记为已完成                     │
  │  4. 删除待办事项                     │
  │  5. 查看统计                         │

请选择操作 (0-5):
```

| `1` | 添加待办事项 | 输入内容 → 选择优先级(高/中/低) → 可选截止日期 |

## 数据存储

所有数据以 JSON 格式持久化保存在 `todos.json` 文件中，与 `todo_manager.py` 同目录。

### 数据格式

```json
[
  {
    "id": 1782279806889,
    "content": "完成项目报告",
    "priority": 3,
    "due_date": "2026-06-30",
    "completed": false,
    "created_at": "2026-06-24 10:30:00"
  }
]
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | `integer` | 唯一标识，基于时间戳自动生成 |
| `content` | `string` | 待办内容 |
| `priority` | `integer` | 优先级：`3`=高, `2`=中, `1`=低 |
| `due_date` | `string` | 截止日期（`YYYY-MM-DD` 格式），可为 `null` |
| `completed` | `boolean` | 完成状态：`true`=已完成, `false`=未完成 |
| `created_at` | `string` | 创建时间（`YYYY-MM-DD HH:MM:SS` 格式） |

## 常见问题

### 问：在 Windows 终端中图标显示乱码怎么办？

Windows 终端默认使用 GBK 编码，可能无法正确显示 emoji 和特殊符号。执行以下命令切换为 UTF-8 编码：

```bash
chcp 65001
```

切换后重新运行程序即可正常显示。

### 问：数据文件在哪里？

数据文件 `todos.json` 自动生成在 `todo_manager.py` 相同的目录下。如需重置数据，直接删除此文件即可。

### 问：如何取消当前输入？

在添加待办事项等输入过程中，按 `Ctrl+C` 可安全取消，返回主菜单。

### 问：为什么删除操作需要二次确认？

为防止误删重要数据，按 ID 删除时会要求输入 `y` 确认后才执行删除。
